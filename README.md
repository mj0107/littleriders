## 📄 기획 배경

어린이 스쿨 버스 관련 안전 사고는 매년 끊임 없이 발생하고 있음 <br>
이를 해결하기 위해 **동승자 탑승 의무화**, **실시간 위치 추적**, **안전운행 여부**를 파악할 수 있는 서비스를 개발하였음.
<br>

- **관리자** 서비스를 통해 어린이가 탑승할 정류장과 노선을 등록하고 관리할 수 있음
- **학부모님**은 원생 승차/하차 시 문자 알림과 내 자녀의 실시간 위치를 확인할 수 있음
- **셔틀버스 기사님**은 운행할 노선 및 정류장 확인과 탑승해야 할 원생의 정보를 확인할 수 있음
  <br>

- **학원 - 셔틀버스 기사님 - 부모님 간의 원생에 대한 정보를 실시간, 유기적으로 확인하여 예기치 못한 사고를 방지하는 것이 목표임**

<br>

## 주요 기술

### 실시간 위치 추적 시스템

- WebSocket을 활용한 실시간 위치 데이터 전송
- MongoDB를 활용한 위치 데이터 저장 및 조회
- Redis를 활용한 실시간 데이터 캐싱
- Spring WebFlux를 통한 비동기 처리

### 안전한 인증 시스템

- JWT 기반의 토큰 인증
- Spring Security를 활용한 보안 처리
- Bcrypt를 통한 비밀번호 암호화
- Redis를 활용한 토큰 관리

### 실시간 알림 시스템

- Spring Mail을 활용한 이메일 알림
- SMS API 연동을 통한 문자 알림
- WebSocket을 통한 실시간 푸시 알림

### 효율적인 데이터 처리

- QueryDSL을 활용한 동적 쿼리 처리
- Spring Data JPA를 활용한 데이터 접근 계층 구현
- MongoDB를 활용한 비정형 데이터 처리
- Redis를 활용한 캐싱 및 세션 관리

### 모던 프론트엔드 아키텍처

- React + TypeScript를 활용한 타입 안정성 확보
- Zustand를 활용한 상태 관리
- Tanstack Query를 활용한 서버 상태 관리
- MSW를 활용한 API 모킹 및 테스트
- Tailwind CSS를 활용한 반응형 디자인

### 하드웨어 연동

- PyQt5를 활용한 디바이스 UI 구현
- BLE(Bluetooth Low Energy) 통신 구현
- GPS 데이터 수집 및 처리
- TTS(Text-to-Speech)를 활용한 음성 안내

### CI/CD 및 배포

- Docker를 활용한 컨테이너화
- Nginx를 활용한 리버스 프록시 및 정적 파일 서빙
- GitLab CI/CD를 활용한 자동화 배포
